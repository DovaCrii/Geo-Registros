# Tooling

Repositorio de apoyo para OpenCode, Codex y skills de IA.

## Contenido
- `tooling/skill-registry.md` — registro local de detección de skills.
- `tooling/skills/` — skills especializadas para revisión, QA, planificación y handoff.

## Uso
Estos archivos documentan cómo operar el repo sin tocar lógica funcional.

## Compatibilidad
La carpeta `.agents/` y `.atl/` pueden quedar vacías como compatibilidad temporal, pero esta es la ubicación canónica nueva.
