# Documentación operador / dron

- **Título:** Documentación operadora y de flota
- **Revisión:** 2026-06-16
- **Fuente oficial:** DGAC Chile
- **Estado:** Borrador interno

## Resumen ejecutivo
Orden mínimo de datos y respaldos para operador y equipo.

## Pasos
1. Verificar credencial del operador.
2. Revisar seguro / vigencia del dron.
3. Confirmar número de serie y trazabilidad.
4. Asociar respaldos al plan.

## Checklist
- [ ] Credencial vigente
- [ ] Número de serie registrado
- [ ] Seguro / vigencia del dron
- [ ] Contacto operativo
