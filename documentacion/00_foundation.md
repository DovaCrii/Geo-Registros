# Foundation Notes

This document records the Block 1 setup direction for Geo-Registros.

## Decisions

- One repository only
- Next.js full-stack for MVP
- PostgreSQL + PostGIS through Docker Compose
- Prisma as ORM foundation
- Local-first document storage prepared under `storage/documents`

## Prepared extension points

- Storage adapter
- Auth/roles scaffold
- Geospatial service scaffold
