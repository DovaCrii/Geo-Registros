# Checklist de permiso de vuelo

- **Título:** Checklist de permiso de vuelo
- **Revisión:** 2026-06-16
- **Fuente oficial:** DGAC Chile
- **Estado:** Borrador interno

## Resumen ejecutivo
Checklist mínima para revisar antes del envío de un plan de vuelo.

## Pasos
1. Completar datos base.
2. Definir área de operación.
3. Cargar documentación.
4. Revisar observaciones.

## Checklist
- [ ] Dron registrado
- [ ] Operador vigente
- [ ] Centro de costo asignado
- [ ] Cliente asignado
- [ ] Área definida
- [ ] Documentos adjuntos

> **Advertencia:** Documento de apoyo interno. No reemplaza la normativa oficial DGAC.
